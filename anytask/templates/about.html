Про AnyTask

Кратко о сервисе

AnyTask — сервис для учета выполнения студентами программы по учебным курсам.

AnyTask поддерживает разные виды курсов: студенты могут самостоятельно выбирать задачи из списка, либо список задач жестко определяется преподавателем.

Например, на курсе Perl есть общий список задач для всех студентов. У каждой задачи есть максимальный балл. Студент может самостоятельно выбирать любую задачу, а преподаватель оценить работу студента. Существует ряд ограничений: у студента не может быть более 2-х неоцененных задач, каждую задачу может выбрать не более 8 человек и если студент взял задачу, то у него есть месяц, чтобы заработать хотя бы один балл. Все это может контролировать AnyTask.

Другой пример — курс Операционные системы. Здесь преподаватель жесто задает список задач для группы и каждый студент должен выполнить все задачи.

AnyTask разработан в ИМКН УрФУ Анной Козловой и Николаем Журавлевым. Идея AnyTask-а отталкивается от сервиса PerlTask, который когда-то давно написал Максим Баклановский.

Немного подробнее

Для преподавателей этот сервис позволяет создавать, редактировать и оценивать задачи их курса. Кроме того, если при оценке работы преподаватель хочет оставить некоторый комментарий или замечание, то это легко сделать с помощью Anytask-a, просто оставив комментарий к оценке.

Студент может брать задачи самостоятельно (если это задано в параметрах курса), либо он должен выполнить некоторый набор задач. Все задачи, относящиеся к студенту и баллы за них можно посмотреть в его личном кабинете.

Кроме того, Anytask хранит список существующих у него курсов и в разделе "Архив" можно посмотреть курсы прошлых лет.

Возможности

Для преподавателя:
  • Быстро создавать задачи.
  • Редактировать задачи прямо со страницы курса.
  • Оставлять комментарий при выставлении балла.
  • Смотреть статистику по группе.
  • Вводить ограничения для курса, такие как:
    • максимальное количество студентов на задачу;
    • "время жизни" заявки студента на задачу без оценки;
    • количество заявок от студента на задачи.
Для студента:
  • Просматривать из профиля список курсов в которых он участвует, а так же список всех его оцененных задач.
  • Отображение времени когда задача была взята и когда истекает срок ее сдачи.
  • Возможность быстро взять задачу, а так же отменить заявку на нее.
  • Просматривать комментарии, оставленных преподавателем при оценке.
